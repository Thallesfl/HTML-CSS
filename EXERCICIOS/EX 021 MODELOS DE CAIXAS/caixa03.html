vamos entender o que significa cada tag indica:
HEADER = SIGNIFICA CABEÇALHO


NAV = O Elemento HTML de Navegação (nav) representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.


MAIN = ONDE SERA FEITO O CONTEÚDO PRINCIPAL


SECTION = A tag (section) no HTML serve para dividir uma página em secções temáticas ou seções de conteúdo relacionadas. Em outras palavras, é usada para agrupar conteúdo que pertence a uma mesma área temática dentro do documento, como um conjunto de posts de blog,-


ARTICLE = A tag (article) em HTML serve para envolver conteúdo independente que poderia ser distribuído em diferentes contextos e ainda fazer sentido por si só


ASIDE = A tag HTML aside serve para delimitar conteúdo que é relacionado, mas não diretamente relevante, ao conteúdo principal da página
